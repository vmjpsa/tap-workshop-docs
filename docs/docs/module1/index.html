    ワークショップ環境の概要 #

    本ワークショップではCode Server と呼ばれるWeb サービスとし提供されるVS Code 環境を利用します。アクセスURL とパスワードに関しては講師からお伝えします。

    ワークショップを提供する環境はこのようになっています。

    TAP は既にインストール済みなので、ワークショップ参加者がTAP のインストールや構成変更をする必要はありません。
